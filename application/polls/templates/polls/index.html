Quarter Round (QR) – principe

Un Quarter Round agit sur 4 mots de 32 bits (a, b, c, d) en combinant :

  • Addition mod 2³² : a = (a + b) mod 2³²
  • XOR : d = d XOR a
  • Rotation gauche : d = ROTL(d, n)

La séquence complète :

  1. a += b; d ^= a; d = ROTL(d,16)
  2. c += d; b ^= c; b = ROTL(b,12)
  3. a += b; d ^= a; d = ROTL(d,8)
  4. c += d; b ^= c; b = ROTL(b,7)

→ Chaque bit de la clé, du compteur ou du nonce influence rapidement tout le bloc.
