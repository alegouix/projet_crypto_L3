# projet_crypto_L3

## Lancer l'application 

### Prérequis 

Django : ```python3 -m pip install django```

### Éxécution

Si nécéssaire, utiliser un environement virtuel : 

```python3 -m venv dev```

```source dev/bin/activate```

Puis éxécuter :

```cd application```

```python3 manage.py runserver```
